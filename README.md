## 🌟 项目特色

### 🎨 核心功能
- **Live2D 动画角色** - 高质量 2D 动画，丰富的表情和动作
- **AI 智能对话** - 支持多种 AI 模型，具备上下文理解能力
- **语音交互** - 语音识别和合成，自然的声音交流

### 🎯 主动互动
- **整点问候** - 每小时提醒休息
- **情感关怀** - 检测负面情绪并给予安慰
- **随机动作** - 不定期展示可爱表情

### 📝 记忆系统
- **短期记忆** - 最近对话记录，用于上下文理解
- **中期记忆** - 会话摘要，支持向量检索
- **长期记忆** - 提取的事实和知识，永久存储

### 💖 心情系统
- **愉悦度** - 影响回复语气的积极程度
- **精力值** - 影响回复长度和活跃度
- **亲密度** - 影响对话的亲密程度
- **无聊度** - 长时间无互动会上升

### 📔 日记功能
- **自动生成** - 每天自动总结对话内容
- **智能摘要** - 提取关键事件和心情变化
- **手动记录** - 支持手动添加日记条目
- **数据导出** - 支持导出日记数据

## 🚀 快速开始

### 环境要求
- Python 3.8+
- Windows 10/11
- 4GB+ 内存
- 支持 OpenGL 3.0+ 的显卡

### 安装运行
```bash
# 克隆项目
git clone https://github.com/SpacervalLam/vivian.git
cd vivian

# 安装依赖
pip install -r requirements.txt

# 运行程序
python main.py
```

## 💖 心情系统详解

### 状态类型
| 状态 | 条件 | 表现 |
|------|------|------|
| 😊 愉悦 | 愉悦>70，精力>50 | 积极活泼的回复 |
| 🎉 兴奋 | 愉悦>80，精力>70 | 非常活跃的互动 |
| 😴 疲惫 | 精力<30 | 简短慵懒的回复 |
| 😴 困倦 | 精力<20，夜间 | 可能进入睡眠 |
| 😐 无聊 | 无聊>70 | 主动发起互动 |
| 😢 难过 | 愉悦<30 | 低沉的语气 |
| 😠 生气 | 愉悦<20 | 冷淡的回复 |

### 状态变化规则
- 每次互动可增加愉悦度和亲密度
- 长时间不互动会增加无聊度、降低精力
- 负面对话会降低愉悦度

## 📔 日记系统

### 日记生成方式
1. **定时生成** - 每天固定时间自动生成
2. **智能触发** - 根据互动频率和重要性自动生成
3. **手动创建** - 用户手动创建日记条目

### 日记内容
- 当日关键事件摘要
- 心情变化曲线
- 互动次数统计
- 关键词标签

## 🔌 插件系统

支持插件扩展，可通过 SDK 开发自定义功能：

```python
from core.sdk.plugin import Plugin

class MyPlugin(Plugin):
    def __init__(self):
        super().__init__("my_plugin", "我的插件", "v1.0")
    
    def on_load(self, context):
        print("插件已加载")
    
    def on_message(self, message):
        return {"text": "插件响应"}
```
